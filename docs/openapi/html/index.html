post /members/{username}/groups/{group_name}/devices
Add a device to a device group. The device UID is a unique identifier for each device. And it's created automatically by the server. A device name must be unique for each user. For example: user_1 can't have two devices named device_1. But user_2 and user_1 can both have a device named device_1. A device name can't contain any of the following characters: / ? < > \ : * | " ' or spaces. (addDeviceToGroup)

Path parameters

username (required)
Path Parameter — The username of the member default: null
group_name (required)
Path Parameter — The name of the device group default: null

post /members/{username}/groups/{group_name}/devices
Add a device to a device group. The device UID is a unique identifier for each device. And it's created automatically by the server. A device name must be unique for each user. For example: user_1 can't have two devices named device_1. But user_2 and user_1 can both have a device named device_1.
